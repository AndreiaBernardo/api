# Desafio API

Este projeto é uma aplicação frontend desenvolvida com **React**, **TypeScript** e **Vite**, focada na integração e consumo de APIS.

## 📸 Preview do Projeto

![Tela do Projeto](./public/tela.png)

## 🚀 Tecnologias Utilizadas

- **React**: Biblioteca principal para a interface.
- **TypeScript**: Tipagem estática para maior segurança e produtividade.
- **Vite**: Ferramenta de build extremamente rápida.
- **ESLint**: Linter para manter a qualidade e padronização do código.

## ✅ Como instalar e rodar o projeto

1. Abra o terminal na pasta do projeto:
   ```bash
   cd "caminho/para/desafio-api"
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse o projeto no navegador:
   ```bash
   http://localhost:5173
   ```

> Se você usa **Yarn**, substitua `npm install` por `yarn` e `npm run dev` por `yarn dev`.
